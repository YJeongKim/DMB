## 💫💬💡

###### `#프로젝트과제` `삽질`

스프링프레임워크 수업의 과제인 일기장 시스템 프로젝트를 셋팅하였다. <br>
수업에서는 Spring Boot + Maven + MyBatis를 STS 툴로 하기 때문에 나도 여기에 따라야하는데, <br>
갓텔리제이로 개발하는거에 익숙하고 훨씬 편하기 때문에 툴만은 양보하지 않기로 했다. <br>
그리고 한참 Gradle + JPA를 공부하다가 다시 Maven + MyBatis를 사용하려니 벌써 막막하였다..😩 <br>
프로젝트의 패키지 구조도 MyBatis를 고려하여 설계를 해야하니 구조를 어떻게 만들어야 하는지 많은 고민을 하였다. <br>
JPA를 무척 사용하고 싶어졌다... 겪어보니 너무 편리한 것..❤ <br>
먼저 프로젝트를 만드는데 Maven으로 Spring Boot 프로젝트를 만드려니 시작부터 꼬이기 시작했다. <br>
STS에서는 Spring Stater Project로 만들면되는데, IntelliJ에서의 Maven + Spring Boot는 기억이 나질 않았다. <br>
그래서 여기서도 약간의 삽집을 해버렸다.ㅎㅎ.. 나중을 위해 각각의 방법을 갑자기 정리해보겠다.ㅋ

#### 1. STS로 Spring Boot + Maven 프로젝트 만들기

1. File/New/Spring Starter Project로 생성

   Spring Starter Project : 스프링 부트를 기반으로 한 프로젝트 <br>
   Spring Legacy Project : 일반 스프링 프레임워크 프로젝트 (개발자가 환경설정, 라이브러리 임의로 해주어야함)

<img src="https://user-images.githubusercontent.com/33328991/82865448-b1b08080-9f61-11ea-843a-0c8e866ce7cd.png" width="75%">

2. Name, Group, Artifact 등 입력사항을 수정하고 Next

<img src="https://user-images.githubusercontent.com/33328991/82865642-2aafd800-9f62-11ea-8b4f-745d84c21df9.png" width="60%">

3. 필요한 라이브러리가 있다면 선택하고 Finish

<br>

#### 2. IntelliJ로 Spring Boot + Maven 프로젝트 만들기

1. File/New/Project 클릭

<img src="https://user-images.githubusercontent.com/33328991/82875798-d52ff700-9f72-11ea-86b2-1e5e39628583.png" width="60%">

2. Spring Initializr Tab 선택 후 Next

<img src="https://user-images.githubusercontent.com/33328991/82875863-ee38a800-9f72-11ea-8879-2c4ee03c79d8.png" width="90%">

3. Group, Artifact Name 등 입력사항을 수정하고 **Type이 Maven Project**로 되어있는지 확인 후 Next <br>

   Maven Project : Maven 프로젝트의 기본 디렉토리 정책에 따르는 구조로 생성한다. (우리가 생각하고 원하는 그 구조이다.) <br>
   Maven POM : pom.xml을 포함한 프로젝트를 생성한다. (호기심에 이걸로 만들었다가 일일이 폴더를 추가해줘야했다.ㅎㅋ)

<img src="https://user-images.githubusercontent.com/33328991/82875992-11635780-9f73-11ea-9ac6-75f133c2aa3f.png" width="90%">

4. 필요한 라이브러리가 있다면 선택하고 Next

5. 입력사항 확인하고 Finish

<br>

정리하느라 진 빼서 급 마무리ㅎㅎ..
